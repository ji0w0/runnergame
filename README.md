# Paint Area Runner

이 게임은 러너 장르에 페인트 영역 계산 시스템을 결합한 프로토타입입니다.

플레이어가 지나간 자리나 발사체가 닿은 위치에 페인트가 남고, 게임은 칠해진 영역의 비율을 계산합니다.

```text
칠해진 면적 ÷ 전체 칠할 수 있는 바닥 면적 × 100 = 점수 %
